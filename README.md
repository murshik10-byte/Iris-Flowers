# Iris Flower Classification
## Project Overview

This project applies machine learning techniques to classify iris flowers into three species — Setosa, Versicolor, and Virginica — based on their physical features.

It is a beginner-friendly project that demonstrates the complete workflow of a supervised learning problem, from data preprocessing to model evaluation.

## Objectives
Understand the Iris dataset
Perform data preprocessing and visualization
Train classification models
Evaluate model performance
Gain hands-on experience with machine learning
##  Dataset
Source: sklearn.datasets.load_iris()
Total Samples: 150
Features:
Sepal Length
Sepal Width
Petal Length
Petal Width
Target Classes:
0 → Setosa
1 → Versicolor
2 → Virginica

## Workflow
1. Data Loading
Loaded dataset using sklearn
Converted into Pandas DataFrame
2. Data Preprocessing
Checked for missing values
Applied feature scaling (if needed)
3. Data Visualization
Plotted graphs to understand feature relationships
Identified patterns between classes
4. Model Building

Algorithms used:

Logistic Regression
K-Nearest Neighbors (KNN)
Decision Tree
5. Model Evaluation
Accuracy Score
Confusion Matrix
Classification Report
## Results
Achieved high accuracy (typically above 90%)
Model effectively classifies iris species
KNN and Logistic Regression perform very well on this dataset
